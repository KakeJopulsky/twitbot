# tweetbot #
Queue up tweets to be posted in the f u t u r e. Want to edit or delete a scheduled tweet? No problem. 

## Getting Started ##
```
npm install
npm run build
npm start
```
## Built with
* Express.js
* React.js
* MongoDB

## Packages
* PassportJS
* node-scheduler
* Twit
